Polished Crystal Save Patcher

This tool allows you to patch your Polished Crystal save files from Version 7 (3.0.0-beta) to Version 8 (9bit). Follow the steps below to patch your save file:

  1. Save your game on the second floor of the Mon Healing Center P.C. (Where you normally do link battles/trades)
  2. Back up your original save file somewhere safe.
  3. Click the "Choose File" button to select your old save file.
  4. Click the "Patch Save" button to start the patching process.
  5. Once the patching is complete, a new save file will be downloaded automatically.
  6. After loading your patched save for the first time, promptly exit the Mon Healing Center P.C. before interacting with anything.
